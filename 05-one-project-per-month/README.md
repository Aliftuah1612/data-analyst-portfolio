# 🚢 One Project per Month

One shipped project every month, rotating across four formats so the portfolio shows breadth *and* consistency:

| Format | What it proves |
|--------|----------------|
| 📊 **A Dashboard** | Visual communication & metric design |
| 🗃️ **A SQL Case Study** | Query craft + business reasoning, end to end |
| 📑 **A Power BI Report** | Modeling (DAX, relationships) + stakeholder-ready reporting |
| 🧹 **A Data Cleaning Project** | The unglamorous 80% of real analyst work, done rigorously |

## Project structure

```
YYYY-MM-project-name/
├── README.md        ← problem, approach, result, what I'd do differently
├── data/            ← source or sample data
├── work/            ← queries / notebook / workbook / .pbix
└── output/          ← final deliverable + screenshots
```

## The rule

**Shipped beats perfect.** Each project has a hard monthly deadline. A finished project with honest limitations documented in its README is worth more than a perfect project that never ships.
